# SwiftLiverpool

![Output](https://raw.githubusercontent.com/slyd0g/SwiftLiverpool/main/example.png)

## Description

This tool leverages the CoreLocation API on macOS to enumerate Location Services data.

## Usage
- Dump global TCC.db as a pseudo table
    - ```./SwiftLiverpool```
